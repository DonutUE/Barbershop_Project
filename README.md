#  Barbershop Premium — Система онлайн-бронювання

Веб-додаток для барбершопу, розроблений на Python (Django). Система дозволяє клієнтам переглядати послуги, обирати майстрів, читати відгуки та бронювати візити онлайн.

## Функціонал

### Для Клієнта:
* **Реєстрація та Авторизація:** Створення особистого кабінету.
* **Онлайн-запис:** Вибір послуги, майстра, локації (Київ/Львів), дати та часу.
* **Особистий кабінет:** Перегляд історії своїх записів зі статусами.
* **Майстри:** Перегляд профілів майстрів з фото, рейтингом та відгуками.
* **Відгуки:** Можливість залишити відгук та поставити оцінку (1-5 зірок) після авторизації.
* **Сортування:** Фільтрація послуг за ціною (від дешевих / від дорогих).
* **Інтерактивна карта:** Відображення локацій на OpenStreetMap.

### Для Адміністратора:
* Зручна адмін-панель (Django Admin).
* Управління записами, користувачами, майстрами та послугами.
* Модерація відгуків.

## Технології

* **Backend:** Python 3, Django 6.0
* **Database:** SQLite (з підтримкою збереження на Google Drive)
* **Frontend:** HTML5, CSS3, Materialize CSS framework
* **Tunneling:** PyNgrok (для доступу до локального сервера з інтернету)
* **Deployment:** Google Colab Environment

## Як запустити проект

Оскільки проект розроблено в середовищі Google Colab:

1.  Відкрийте файл `Barbershop_Project.ipynb` у Google Colab.
2.  Вставте свій токен `ngrok` у відповідну змінну в коді.
3.  Запустіть клітинку з кодом.
4.  Перейдіть за посиланням, яке згенерує `ngrok` (наприклад, `https://xxxx.ngrok-free.dev`).
